# AWS Agentic AI Central - Project Overview

This repository contains the content for the Agentic AI Central website, a hub to aggregate and search resources on agentic AI, and how to implement it with AWS.

## Site Structure

The site is structured in a way that customers visiting it can:

1. **Discover**: Having service-agnostic and easy to understand design patterns for agentic AI, and getting started guides for AWS services for agentic AI.

2. **Learn**: Where they can quickly search for up-to-date AWS blogs, videos (in YouTube) and other relevant resources, as well as complete AWS workshops on specific topics or services.

3. **Build**: With blueprints' guides for easy step by step implementation of common patterns, and full repositories with code samples for specific use cases, services or solutions.

## Content Guidelines

- The site is inspired by AWS Serverlessland (https://serverlessland.com/)
- Content should be easy to read and digest by readers with any level of knowledge
- Use direct and practical language
- Keep content length appropriate and focused
- This site is owned by Amazon Bedrock GTM Specialist Solutions Architects in AWS
- Supported by contributions from all the AWS community including customers and AWS technical roles
