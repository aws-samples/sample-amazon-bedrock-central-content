# Amazon Bedrock

Amazon Bedrock is a fully managed service that makes high-performing foundation models (FMs) from leading AI companies and Amazon available through a unified API. You can choose from a wide range of foundation models to find the one best suited for your use case, privately customize them with your data using techniques such as fine-tuning and Retrieval Augmented Generation (RAG), and build agents that execute tasks using your enterprise systems and data sources.

With Amazon Bedrock's serverless experience, you can get started quickly without managing any infrastructure.

## Getting Started

1. **Sign up for AWS**: If you don't have an AWS account, create one at [aws.amazon.com](https://aws.amazon.com)
2. **Set up IAM permissions**: Create a role with the `AmazonBedrockFullAccess` managed policy
3. **Enable model access**: Foundation models are enabled by default since June 2025. For Anthropic models, first-time users may need to submit use case details
4. **Explore the console**: Use the Playgrounds in the Amazon Bedrock console to run prompts and generate responses
5. **Use the API**: Set up access through the Amazon Bedrock API or supported SDKs

## Key Features

- **Choice of Foundation Models**: Access to Claude, Llama, Mistral, Amazon Titan, and more
- **Serverless Experience**: No infrastructure to manage
- **Security & Privacy**: Your data is not used to train the underlying models
- **Agents & Knowledge Bases**: Build agents that can reason, plan, and execute tasks
- **Guardrails**: Built-in safeguards for responsible AI

## Use Cases

- Content generation and summarization
- Conversational AI and chatbots
- Text and code generation
- Search and personalization
- Agentic workflows and automation
